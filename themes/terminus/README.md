# terminus-hugo-template

基于 [terminus-jekyll-template](https://github.com/TerminusBot/terminus-jekyll-template)。如要从 Jekyll 迁移到 Hugo，只需将原 `_posts/` 中的文件移到 `content/posts/`，并配置好 `config.yml` 即可。

## 使用指南

无
